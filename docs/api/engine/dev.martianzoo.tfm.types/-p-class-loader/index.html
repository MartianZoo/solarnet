class PClassLoader(authority: Authority, autoLoadDependencies: Boolean = false)
Content copied to clipboard

All PClass instances come from here. Uses an Authority to pull class declarations from as needed. Can be frozen, which prevents additional classes from being loaded, and enables features such as PClass.allSubclasses to work.

Parameters

authority

the source of class declarations to use as needed; loadEverything will load every class found here

autoLoadDependencies

whether, when a class is loaded, to also load any classes that class depends on in some way; by default only superclasses are auto-loaded
