Everything there is to know about a Terraforming Mars card, except for text (including the card name). It's theoretically possible to reconstruct acceptable instruction text from this data, just not the original wording.

Validation: the properties state numerous "guarantees", but in most cases this class is not able to internally validate those conditions. Actual parsing of the strings must be handled by a PETS parser external to this library.
