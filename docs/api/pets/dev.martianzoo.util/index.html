interface Grid<E> : Set<E>
Content copied to clipboard

A fixed-size two-dimensional array of nullable elements, where the elements "know" their own row and column number (as opposed to their being placed there). Important: there is no distinction made between a null cell, a missing cell, and a cell that is "off the edge" of the grid!
