abstract class CustomInstruction(val functionName: String)
Content copied to clipboard

For instructions that can't be expressed in Pets, write @functionName(Arg1, Arg2...) and implement this interface. Any Authority providing a class declaration that uses that syntax will need to also return this instance from Authority.customInstruction.

Only one of translate or execute need be overridden.
